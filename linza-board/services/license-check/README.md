# License Check

Сервис проверки валидности лицензий платформы **Linza Detector**.

## Ответственность

- Валидация лицензионных ключей
- Проверка сроков действия
- Проверка лимитов использования
- Кэширование результатов проверки (Redis)

## API

| Метод | Эндпоинт | Описание |
|-------|----------|----------|
| POST | `/check` | Проверка лицензии |
| GET | `/check/{key}` | Проверка по ключу |
| GET | `/health` | Проверка состояния |
