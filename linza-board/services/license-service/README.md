# License Service

Сервис управления лицензиями платформы **Linza Detector**.

## Ответственность

- CRUD операции с лицензиями
- Генерация лицензионных ключей
- Привязка лицензий к организациям/пользователям
- Управление сроками действия лицензий

## API

| Метод | Эндпоинт | Описание |
|-------|----------|----------|
| GET | `/licenses` | Список лицензий |
| POST | `/licenses` | Создание лицензии |
| GET | `/licenses/{id}` | Получение лицензии |
| PUT | `/licenses/{id}` | Обновление лицензии |
| DELETE | `/licenses/{id}` | Удаление лицензии |
| GET | `/health` | Проверка состояния |
