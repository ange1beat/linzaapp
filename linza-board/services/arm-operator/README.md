# АРМ Оператора

Автоматизированное рабочее место оператора платформы **Linza Detector**.

## Ответственность

- Мониторинг процессов обработки видео
- Управление пользователями и организациями
- Администрирование лицензий
- Просмотр логов и диагностика
- Модерация обратной связи

## Технологии

- Vue.js 3 + Vite
- Pinia (state management)
- Vue Router
- Axios (HTTP-клиент)
