# АРМ Клиента

Автоматизированное рабочее место клиента платформы **Linza Detector**.

## Ответственность

- Интерфейс для загрузки видео на анализ
- Просмотр результатов анализа
- Управление задачами
- Управление лицензиями
- Просмотр статистики и отчётов

## Технологии

- Vue.js 3 + Vite
- Pinia (state management)
- Vue Router
- Axios (HTTP-клиент)
